# Finding Mods

Where to find and download Roblox mods for AppleBlox.

## Official Sources

### GameBanana

**Primary mod repository for Roblox**

- Hundreds of texture packs and themes
- User ratings and reviews
- Regular updates
- Safe and moderated

**Link:** [gamebanana.com/games/2879](https://gamebanana.com/games/2879)

### AppleBlox Discord

**Community-shared mods**

- Get recommendations
- Share your own creations
- Latest mod releases
- Direct support

**Link:** [appleblox.com/discord](https://appleblox.com/discord)

### Bloxstrap Community

**Bloxstrap mods work with AppleBlox!**

- Large existing mod library
- Same file structure
- Full compatibility

**Link:** [github.com/pizzaboxer/bloxstrap/wiki/Adding-custom-mods](https://github.com/pizzaboxer/bloxstrap/wiki/Adding-custom-mods)

## Mod Compatibility

### Bloxstrap Mods

- ✅ Fully compatible with AppleBlox
- ✅ Same file structure and system
- ✅ Can use Bloxstrap mods without changes

### Retina vs Non-Retina

**Display Compatibility:**
- Some mods are designed for lower resolutions
- May appear blurry or too sharp on Retina displays
- Enable **"Legacy Resolution"** in Mods panel if needed
- Modern mods usually support high-DPI displays

### Game-Specific Mods

- Some mods are designed for specific games
- Universal mods work across all games
- Check mod description for compatibility info

## What to Look For

### Quality Indicators

When browsing mods:

1. **High Ratings** - User feedback is important
2. **Recent Updates** - Active maintenance
3. **Clear Description** - Good documentation
4. **Screenshots** - See before downloading
5. **File Size** - Reasonable size (not suspiciously small/large)

### Safety Considerations

::: warning Mod Safety
- Only download from trusted sources
- Avoid mods from unknown websites
- Check file contents before installing
- Scan downloads with antivirus if concerned
:::

**Safe Sources:**
- ✅ GameBanana
- ✅ AppleBlox Discord (verified uploads)
- ✅ Bloxstrap Wiki
- ✅ GitHub (verified repositories)

**Unsafe Sources:**
- ❌ Random websites
- ❌ Suspicious Discord DMs
- ❌ Unverified file sharing sites

## Downloading Mods

### From GameBanana

1. Visit [gamebanana.com/games/2879](https://gamebanana.com/games/2879)
2. Browse or search for mods
3. Click on a mod you like
4. Click **"Download"** button
5. Save the ZIP file to your Downloads folder

### From Discord

1. Join [AppleBlox Discord](https://appleblox.com/discord)
2. Go to `#mods` channel
3. Browse shared mods
4. Download attachments or follow links
5. Save to Downloads folder

### File Format

Mods are usually distributed as:
- **.zip** - Compressed archive (most common)
- **.7z** - 7-Zip archive (extract with The Unarchiver)
- **.rar** - RAR archive (extract with The Unarchiver)
- **Folder** - Sometimes direct folder downloads

::: tip Extracting Archives
macOS can extract .zip files natively. For .7z and .rar, install [The Unarchiver](https://theunarchiver.com/) from the App Store.
:::

## Mod Recommendations

### For Beginners

Start with these popular, stable mods:

- **Minimal UI** - Clean, simple interface
- **HD Textures** - Enhanced visual quality
- **Dark Theme** - Eye-friendly dark mode

### For Performance

If you have an older Mac:

- **Low-Res Textures** - Smaller file sizes
- **Optimized Packs** - Performance-focused
- **Minimal Mods** - Fewer files to load

### For Customization

If you want full control:

- **Modular Packs** - Mix and match components
- **Themed Collections** - Seasonal or game-specific
- **Custom Icons** - Personalized buttons and UI
