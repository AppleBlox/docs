# Misc & System

Found on the **Misc** page.

## Settings

- **Notification Sounds** - Play sounds with macOS notifications.
- **Alternative Notifications** - Use AppleScript-based notifications instead of the native alerter binary. Enable this if default notifications are buggy for you.
- **Fixed Loading Times** - Keep the loading screen visible longer so you can read status messages.

## Actions

- **View Logs** - Open the logs folder in Finder.
- **Copy Debug Report** - Copy a structured report to your clipboard with system info, active settings, and recent errors. Paste it directly into a bug report.
- **Export Debug Bundle** - Save your logs, configuration, and debug summary as a `.zip` archive. Useful when reporting bugs on Discord or GitHub. Also triggered by `Cmd+P` from anywhere in the app.
- **AppleBlox Folder** - Jump to the AppleBlox data directory in Finder.
- **Reset AppleBlox** - Completely wipe all data, settings, and your stored cookie. This is a destructive action with a two-step confirmation. You'll need to type "RESET" to proceed.

::: danger Reset
Resetting AppleBlox permanently deletes all settings, mods configuration, cached data, and your stored Roblox cookie. This cannot be undone.
:::
