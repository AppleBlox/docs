# Quick Start Guide

Get up and running with AppleBlox in 5 minutes!

## Prerequisites

- macOS 11 (Big Sur) or later
- Internet connection
- Roblox account

## Installation Steps

### 1. Download AppleBlox

Visit the [releases page](https://github.com/AppleBlox/appleblox/releases/latest) and download the appropriate version:

- `AppleBlox-arm64.dmg` for Apple Silicon (M1/M2/M3/M4)
- `AppleBlox-x64.dmg` for Intel Macs
- `AppleBlox-universal.dmg` for either architecture

::: tip How to check your Mac's architecture
Click the Apple menu > About This Mac. If you see "M1", "M2", "M3", or "M4", you have Apple Silicon. If you see "Intel Core i5" or similar, you have an Intel Mac.
:::

### 2. Install the App

1. Open the downloaded DMG file
2. Drag **AppleBlox.app** to your Applications folder
3. Close the DMG window
4. Eject the disk image (right-click > Eject)

### 3. First Launch

1. Open AppleBlox from Applications
2. Grant any permission requests (accessibility, etc.)
3. The app will initialize its data directory automatically

::: warning Security Warning
If you see "AppleBlox is an app downloaded from the internet", click **Open**. This is a normal macOS security warning for downloaded apps.
:::

### 4. Install Roblox (if needed)

If Roblox isn't installed:

1. The sidebar button will show **"Install"**
2. Click the Install button
3. AppleBlox will download and install Roblox for you
4. Progress is shown with download percentage and speed

### 5. Launch Your First Game

1. Once Roblox is installed, the button changes to **"Play"**
2. Click Play to launch Roblox
3. Visit roblox.com and join any game
4. AppleBlox handles the deeplink automatically!

## Sidebar Button States

The sidebar button shows different states:

| State | Color | Description |
|-------|-------|-------------|
| **Install** | Blue | Roblox not installed |
| **Play** | Green | Ready to launch |
| **Active** | Teal | Roblox is running |
| **Kill** | Red (on hover) | Force close Roblox |
